# 🐕 STM32 桌面电子宠物小狗

基于 STM32F103C8T6 的四足仿生桌宠，支持蓝牙遥控、16 种动作、OLED 表情切换、呼吸灯效果。

## 📖 项目简介

这是一只可以放在桌面的电子宠物小狗，使用 **5 个舵机**（四条腿 + 尾巴）实现各种动作，**0.96 寸 OLED** 显示多种表情（开心、睡觉、瞪眼等），通过**蓝牙**接收手机指令，还可以控制 LED 呼吸灯光。代码完全开源，适合 STM32 初学者学习舵机控制、PWM、串口通信和 OLED 驱动。

---

## ✨ 功能特性

- 🕹️ **蓝牙遥控**：手机 APP 发送十六进制指令，控制小狗动作
- 🐾 **16 种动作**：前进、后退、左转/右转、摇摆、跳跃、摇尾巴、打招呼、伸懒腰等
- 😀 **表情系统**：7 种 OLED 表情（快乐、睡眠、瞪眼、狂热等）
- 💡 **灯光效果**：支持 LED 常亮 / 呼吸灯模式
- 🗣️ **语音预留**：可扩展语音模块（串口指令相同）
- 🔧 **可调速**：蓝牙指令可动态调整移动速度和摇摆速度

## 🧰 硬件清单

| 组件 | 型号 / 规格 | 数量 |
|------|-------------|------|
| 主控 | STM32F103C8T6 最小系统板 | 1 |
| 舵机 | SG90 / MG90S（5V） | 5 |
| OLED | 0.96 寸，I2C 接口（SSD1306） | 1 |
| 蓝牙模块 | HC-05 / HC-06 | 1 |
| LED | 直插或贴片（颜色自定） | 2 |
| 电阻 | 220Ω（LED 限流） | 2 |
| 电源 | 5V / 2A 移动电源 或 18650+升压板 | 1 |
| 下载器 | ST-Link V2 | 1 |
| 杜邦线、面包板 | 若干 | - |

> ⚠️ **舵机供电**：必须使用 **5V/2A 外部电源**，不可从 STM32 的 3.3V 取电，否则会导致复位或舵机无力。

## 🔌 引脚连接表

| 功能 | STM32 引脚 | 备注 |
|------|------------|------|
| 左上腿舵机 | PA0 (TIM2_CH1) | `Servo_Angle1` |
| 右上腿舵机 | PA1 (TIM2_CH2) | `Servo_Angle2`（内部做了 180-Angle 映射）|
| 左下腿舵机 | PA2 (TIM2_CH3) | `Servo_Angle3` |
| 右下腿舵机 | PA3 (TIM2_CH4) | `Servo_Angle4`（内部做了 180-Angle 映射）|
| 尾巴舵机 | **PB0** (TIM3_CH1) | `WServo_Angle` |
| LED1（左眼） | **PB1** (TIM3_CH3) | 与 LED2 可能共用引脚（见下方说明） |
| LED2（右眼） | **PB1** (TIM3_CH4) | 实际与 LED1 相同引脚，建议只接一个 LED 或修改代码 |
| OLED SCL | PB8 | I2C 时钟 |
| OLED SDA | PB9 | I2C 数据 |
| 蓝牙 TX → PB11 | PB11 (USART3_RX) | 接蓝牙模块的 TX |
| 蓝牙 RX → PB10 | PB10 (USART3_TX) | 接蓝牙模块的 RX |
| 语音 TX（预留） | PA9 (USART1_TX) | 接语音模块的 RX |
| 语音 RX（预留） | PA10 (USART1_RX) | 接语音模块的 TX |
| GND | 所有模块共地 | |
| 舵机供电（5V） | 外接 5V 电源 | 与 STM32 共地即可 |
| STM32 供电 | 3.3V (ST-Link 或 USB 转 TTL) | |

> **⚠️ 引脚冲突说明**：  
> 代码中 `PWM_LED1` 使用 TIM3_CH3（默认 PB0），`PWM_LED2` 使用 TIM3_CH4（默认 PB1）。  
> 但是 `PWM_Init` 中只开启了 GPIOB_Pin_0 和 Pin_1，而尾巴舵机使用了 TIM3_CH1（PB0）。  
> 所以 **LED1 与尾巴舵机共用 PB0**，**LED2 单独使用 PB1**。  
> 解决方法见“常见问题”章节。
